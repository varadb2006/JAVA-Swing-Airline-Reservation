# Airline Reservation System ✈️

A desktop-based Airline Reservation System developed using **Java Swing** and **MySQL** as part of a hands-on learning project to understand **frontend and backend integration in Java**.

This project focuses on implementing real-world reservation workflows while strengthening concepts such as database connectivity, UI design, and CRUD operations.

---

##  About the Project

The Airline Reservation System is a Java desktop application that simulates the core functionalities of an airline booking platform.  
It allows administrators and users to manage customers, flights, and reservations through a graphical user interface.

This project was built primarily for **learning purposes** and to gain practical experience in:
- Java Swing UI development
- MySQL database integration
- JDBC connectivity
- Application workflow design
